# Prompt App

## Pre-requisites

1. Python 3.12 or higher
2. Poetry

## Installation

1. Clone the project repository.
2. Initialize the virtual environment and install dependencies via:
   
   ```bash
   poetry install
   ```

3. Run the Project

    ```bash
    python promptapp/app.py
    ```

## Visit in Browser
http://127.0.0.1:5000
